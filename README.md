Sales Analysis Project using PySpark &amp; Databricks
---------------------------------------------------------------------
🚀 Project Overview

This project focuses on building an end-to-end sales data analysis pipeline using PySpark on Databricks. It demonstrates how large-scale transactional data can be processed, transformed, and analyzed efficiently to generate actionable business insights.

The project simulates a real-world data engineering + analytics workflow, including data ingestion, transformation, aggregation, and visualization-ready outputs.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🎯 Objectives

Perform scalable data processing using PySpark
Clean and transform raw sales data
Generate KPIs and business metrics

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🏗️ Architecture

Raw Data (CSV)
        ↓
Data Ingestion (Databricks / DBFS)
        ↓
Data Cleaning & Transformation (PySpark)
        ↓
Business Aggregations
        ↓
Final Data Output (Tables)
        ↓
Visualization (Databricks)

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🛠️ Tech Stack

PySpark – Distributed data processing
Databricks – Unified analytics platform
Python – Core programming language

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

📈 Key Insights Generated

Identified high-revenue products and categories
Country wise sales distribution trends
monthly/yearly/quarterly sales analysis
Customer visit frequency
sales as per category/product name
